## 中文介绍

01MVP 是 Jackie 维护的 AI 产品实战手册，关注一个人或小团队如何把模糊想法变成能上线、可验证、可收费、可迭代的小产品。

它不是单纯的工具列表，也不是一次性写完的课程。01MVP 更像一个持续更新的产品工作台：从自我定位、想法验证、MVP 范围控制、AI 辅助开发，到上线、收款、获客、运营和复盘。

你可以在 [01mvp.com](https://01mvp.com) 阅读主站内容。主站源码目前不开放，本仓库只作为 01MVP 的公开介绍和入口。其他开源项目可以在 [github.com/01mvp](https://github.com/01mvp) 查看。

### 适合谁

- 有想法，但经常卡在开始之前的人
- 想用 AI 做出自己的小产品的人
- 独立开发者、创作者、产品经理、设计师，以及正在学习 AI 编程的人
- 不只想学工具，也想理解从想法到产品的完整过程的人

### 你可以在 01mvp.com 找到

- [01MVP 实战手册](https://01mvp.com/docs/guide)：从想法、验证、构建、上线到运营的主线路径
- [产品创作思考](https://01mvp.com/docs/product-notes)：关于小产品、独立开发和产品判断的长期笔记
- [AI 工具教程](https://01mvp.com/docs/tools)：AI 模型、工具、工作流和成本相关实践
- [实战案例](https://01mvp.com/docs/cases)：真实项目从需求到上线的拆解
- [模板指南](https://01mvp.com/docs/template)：生产级全栈模板的能力说明和使用方式
- [Skills 手册](https://01mvp.com/docs/skills)：Agent Skill 的安装、使用和组织方法

## English

01MVP is Jackie’s practical handbook for building AI-assisted products. It focuses on how solo builders and small teams can turn rough ideas into small products that can be launched, validated, sold, and improved.

It is not just a list of tools, and it is not a finished course. 01MVP is a living product workbench covering self-discovery, idea validation, MVP scoping, AI-assisted development, launch, payments, user acquisition, operations, and review.

You can read the main content at [01mvp.com](https://01mvp.com). The source code of the 01mvp.com website is not open source at the moment. This repository is a public introduction and a pointer to the website. Other open-source projects are available in the [01MVP GitHub organization](https://github.com/01mvp).

### Who it is for

- People with many ideas who often get stuck before starting
- Builders who want to use AI to ship their own small products
- Indie hackers, creators, product managers, designers, and people learning AI coding
- Readers who want the full path from idea to product, not only tool tutorials

### What you can find on 01mvp.com

- [01MVP Handbook](https://01mvp.com/docs/guide): the main path from idea, validation, building, launch, and operations
- [Product Notes](https://01mvp.com/docs/product-notes): long-running notes about small products, indie building, and product judgment
- [AI Tools](https://01mvp.com/docs/tools): practical notes on AI models, tools, workflows, and costs
- [Case Studies](https://01mvp.com/docs/cases): real project breakdowns from requirements to launch
- [Template Guide](https://01mvp.com/docs/template): capabilities and usage notes for the production-ready full-stack template
- [Skills Handbook](https://01mvp.com/docs/skills): how to install, use, and organize Agent Skills
